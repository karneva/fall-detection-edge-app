# Documentation Index

이 디렉터리는 이 프로젝트를 이해하고 운영하는 데 필요한 핵심 문서만 남긴 상태입니다.

## 문서 구성

- `ARCHITECTURE.md`
  애플리케이션 구조, 실행 모드, 데이터 흐름, 주요 모듈 역할
- `FALL_DETECTION_CORE.md`
  낙상 감지 알고리즘, 상태 전이, 2차 AI 검증 구조
- `OPERATIONS.md`
  실행 방법, API, 런타임 동작, 저장 파일, 운영 시 확인 포인트
- `TUNING_GUIDE.md`
  현재 코드 기준 파라미터와 튜닝 방향

## 권장 읽기 순서

1. `ARCHITECTURE.md`

- 실행/운영에 바로 쓸 수 있는 정보 우선
